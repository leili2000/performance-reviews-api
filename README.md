# performance-reviews-api
Employee Performance Reviews Database API

Created by:
Leili Nelson (lnelso39@calpoly.edu)
Sumedha Kundurthi (skundurt@calpoly.edu)
Pihu Jha (pijha@calpoly.edu)

Using this API, employee performance can be tracked. All employees will be stored in a table. Comments on employees will be saved in another table. Additionally, there will be a table for periodic employee performance review documents.
